# Parallel Companion

Use this companion when `omc-parallel` needs more orchestration discipline.

## Orchestration quality bar

- Define ownership boundaries before delegation.
- Keep delegated tasks bounded and verifiable.
- Keep the parent lane on integration and the critical path.
- Verify the integrated result, not just slice-local output.

## Common failure modes

- Delegating before success is defined
- Multiple lanes writing the same files
- Passive parent integration
