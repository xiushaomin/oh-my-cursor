# Brainstorm Companion

Use this companion when `omc-brainstorm` needs a little more structure than the core skill body should carry.

## Option quality bar

- Make options meaningfully different, not minor variants.
- Tie each option to a concrete user or engineering tradeoff.
- Prefer options that can turn into a plan-ready slice.

## Good endings

- A recommended direction with 1-2 reasons
- A short `plan_seed`
- The smallest unresolved decision if the path is still blocked
