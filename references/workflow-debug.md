# Debug Companion

Use this companion when `omc-debug` needs a tighter investigation loop.

## Debug quality bar

- State the symptom and best available repro early.
- Rank a small set of plausible causes before editing.
- Gather evidence until one cause is supported.
- Re-run the proof after the fix.

## Common failure modes

- Guessing without a repro path
- Multi-variable fixes
- Cleanup work disguised as debugging
