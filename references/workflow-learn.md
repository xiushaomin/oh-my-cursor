# Learn Companion

Use this companion when `omc-learn` should produce a stronger prevention artifact.

## Learning quality bar

- Separate facts from inference.
- Make the lesson reusable, not just narrative.
- Prefer durable assets over one-off summaries.
- End with explicit follow-up or ownership.

## Common failure modes

- Slogan-level prevention
- No concrete asset target
- Recreating incident ceremony instead of improving the repo
